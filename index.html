I accomplished my goal of leaving Wordpress today! It was a hard challenge but my blog is now powered by Jekyll and hosted on Github.

Under construction! This site is coded by hand with no frameworks. I'm ready to start learning new skills so I can make this blog awesome.

Why Jekyll?

  • No more heavy CMS, database or FTP to deal with.
  • Hosting is free using Github pages! I now get version control and deployment in one place.
  • I get direct control of my markup. No need for a rich-text editor to write posts.
  • Super fast loading! (5 seconds to 300 ms)
