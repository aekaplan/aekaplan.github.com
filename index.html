I accomplished my goal of leaving Wordpress today! It was a hard challenge but this blog is now powered by Jekyll and hosted on Github.

Under construction! This site is coded by hand with no frameworks. I'm ready to learn new skills so I can make this blog awesome.

Why Jekyll?

  • No more heavy CMS, database or FTP to deal with.
  • Hosting is free using Github pages! I now get version control and deployment in one place.
  • I get to write my blog posts using HTML in the my favorite editor. No more clunky rich-text editor.
  • Super fast loading time!
