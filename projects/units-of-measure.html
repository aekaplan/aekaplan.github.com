CSS Units of measure

  • px
  • em
  • rem
  • %

px

Pixels are fixed-size units and fall into a type of measurement called absolute lengths.

Absolute lengths: are fixed in relation to each other and anchored to some physical measurement. Other absolute length units include mm, cmm and inches.

use for:

  • fixed widths and heights
  • setting a min/max-widths
  • setting media queries
  • borders, box and text shadows

em

An em is a unit of measure that is relative to the current font size and considered a relative length.

Relative lengths: are units that specify a length relative to another length. Other relative length units include rem, percents and vw/vh.

The em unit is relative to the font-size of the parent. For example, 1em is equal to 100% of the base font size, so 1.5em is 150% etc.

use for:

  • easily changing your base font size in one place
  • flexible responsive web design
  • reserving an amount of space relative to your current font-size
  • maintaing the relationship between font size and margin/padding

rem

The rem or "root em unit" bases it's font size off the root HTML element instead of nesting and multiplying font sizes (em).

One of the issues with using ems for font sizing is that they compound as they nest. For example, a list within a list will multiply in size as you continue to nest. The rem units are constant throughout your html.

%

The percent (%) unit is very much like an em and is considered a relative length.

Percents scale font sizes and make widths on divs and containers fluid. It's the building block for responsive web design where everything needs to be dynamic.

use for:

  • fluid widths and heights
  • scalable font-sizes
  • padding and margins

vw

Responsive Header: resize your browser!

The text above uses a new unit called vw which scales the font size based off the viewport width. These units would help eliminate the need to size fonts using media queries.

  • 1vw = 1% of viewport width
  • 1vh = 1% of viewport height
  • 1vmin = 1vw or 1vh, whichever is smaller

Pretty cool right? Browser support is too weak to use these units on live sites. Check out this tutorial to learn more.

:)

Relative units work best with responsive design but absolute units have their place.

By using a combination of percents, ems/rems and pixels, you can achieve a layout that is both beautiful and fluid. Responsive web design is good design!
